 Amazon Sales Analysis SQL Project

This project contains SQL queries for analyzing Amazon/E-commerce sales data using MySQL.

## KPIs Covered
- Total Sales
- Total Orders
- Average Order Value
- Average Review Score

## Analysis Performed
- Order Status Analysis
- Payment Type Analysis
- Monthly Sales Trend
- Customer Spending Analysis
- Product Revenue Analysis
- Shipping Cost Analysis
- Delivery Performance Analysis
- Review Distribution Analysis

## SQL Concepts Used
- JOIN
- GROUP BY
- ORDER BY
- Aggregate Functions
- HAVING Clause
- DATEDIFF
- MONTHNAME
- Aliases

## Tools Used
- MySQL
- GitHub

## Project Type
Data Analytics SQL Project
